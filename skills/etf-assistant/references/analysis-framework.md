# ETF 专业分析框架

本文档提供 ETF 分析的方法论和指标解读指南。在进行深度分析时参考。

## 一、多维度分析体系

ETF 分析不应只看"涨了还是跌了"。专业分析至少覆盖以下五个维度：

### 1. 收益维度 — 不同时间窗口的回报

| 周期 | 用途 |
|------|------|
| 近1周 | 短线情绪和资金动向 |
| 近1月 | 短期趋势确认 |
| 近3月 | 中期趋势判断 |
| 近6月 | 半年度表现，排除短期噪音 |
| 近1年 | 长期趋势，适合做配置决策 |
| YTD（年初至今） | 当年业绩评估 |

**解读要点**：
- 短期收益高但长期收益低 → 可能是反弹而非反转
- 长期收益高但短期回调 → 可能是买入机会
- 所有周期都在下跌 → 趋势极差，不宜抄底
- 短、中、长期收益递增 → 健康的上升趋势

### 2. 风险维度 — 波动与回撤

**年化波动率**（Annualized Volatility）
- 计算方法：日收益率标准差 × √252
- < 15%：低波动（债券型、红利型）
- 15-25%：中等波动（宽基指数）
- 25-35%：高波动（行业主题）
- > 35%：极高波动（杠杆/小众主题）

**最大回撤**（Maximum Drawdown）
- 从峰值到谷底的最大跌幅
- 反映"最坏情况"下你会亏多少
- 宽基指数典型回撤：20-40%
- 行业ETF典型回撤：30-60%
- 你能承受的最大回撤决定了你的仓位上限

**当前回撤**
- 从近期高点到当前价的跌幅
- 回撤较大时可能是逢低布局的时机（但需结合趋势判断）

**夏普比率**（Sharpe Ratio）
- 计算方法：（年化收益 - 无风险利率）/ 年化波动率
- < 0：负收益，不如存银行
- 0-0.5：较差的风险调整收益
- 0.5-1.0：可接受
- > 1.0：优秀的风险调整收益
- > 2.0：极优秀（长期维持很难）

**卡玛比率**（Calmar Ratio）
- 计算方法：年化收益 / 最大回撤
- 衡量每承受一单位最大回撤所获得的收益
- > 1.0 表示收益能覆盖最大回撤

### 3. 趋势维度 — 均线与动量

**均线系统**
- 5日均线：超短期趋势（日内交易者关注）
- 20日均线（月线）：短期趋势分界线
- 60日均线（季线）：中期趋势分界线
- 120日均线（半年线）：长期趋势分界线
- 250日均线（年线）：牛熊分界线

**均线排列判断**：
- 多头排列（价格 > MA20 > MA60）：上升趋势，持有或加仓
- 空头排列（价格 < MA20 < MA60）：下降趋势，回避或减仓
- 均线纠缠：方向不明，观望为主

**价格位置**：
- 当前价格在52周区间中的位置
- < 20%：接近低点，可能超卖
- 20-40%：低位区间
- 40-60%：中位区间
- 60-80%：高位区间
- > 80%：接近高点，可能超买

### 4. 成交量维度 — 资金流向信号

**量比**（当日成交量 / 20日均量）
- > 2.0：显著放量，有重大事件或资金异动
- 1.2-2.0：温和放量
- 0.8-1.2：量能平稳
- < 0.5：明显缩量

**量价配合分析**：
- 放量上涨：资金流入，趋势确认
- 缩量上涨：上涨乏力，警惕回调
- 放量下跌：恐慌抛售，短期有底但中期看空
- 缩量下跌：下跌动能衰减，可能接近底部
- 放量横盘：多空分歧大，关注方向选择

### 5. 组合维度 — 配置与再平衡

**仓位管理原则**：
- 单只ETF不超过总仓位的30%
- 同行业ETF合计不超过40%
- 至少保留20%现金或债券类资产

**分散化检查**：
- 是否跨风格（大盘 + 中盘 + 小盘）
- 是否跨行业（科技 + 消费 + 医药 + 金融）
- 是否有防御品种（黄金 + 红利）
- 是否有不同板块（主板 + 创业板 + 科创板）

## 二、定投策略深度指南

### 定投的本质
定投利用"波动"而非害怕波动。高波动ETF的定投效果通常优于低波动ETF（前提是长期向上）。

### 定投选择标准
1. **长期上涨确定性高**：选择宽基指数而非单一行业
2. **波动率适中偏高**：波动越大，微笑曲线效应越明显
3. **估值合理**：高估时可以减少定投金额，低估时加码

### 定投增强策略
- **估值定投**：PE百分位低时多投，高时少投或停投
- **均线偏离定投**：价格低于均线时加倍，高于均线时减半
- **止盈策略**：累计收益达30-50%时分批止盈，而非一直持有

### 定投常见误区
- 只定投一个月就放弃 → 定投至少坚持3年
- 大跌时停止定投 → 大跌恰恰是定投最有效的时候
- 只投一只ETF → 应该2-3只分散
- 不设止盈 → 盈利全部回吐

## 三、分析报告输出规范

### 快速查询（价格/行情）
简短展示核心数据，附一句趋势判断。

### 单只ETF深度分析
按以下顺序组织：
1. **一句话结论**：先给判断（如"当前处于低位反弹初期，可小仓位试探"）
2. **核心数据表**：价格、涨跌、波动率、回撤
3. **趋势研判**：均线、量价、位置分析
4. **风险提示**：当前主要风险是什么
5. **操作建议**：具体可执行的建议

### ETF对比分析
使用表格对比关键指标，明确推荐哪只以及为什么。

### 组合诊断
分析持仓的集中度、相关性、风险敞口，给出调整建议。

## 四、分析注意事项

1. **不做精确预测**：说"可能"而非"一定"，给区间而非点位
2. **区分事实和观点**：数据是事实，趋势判断是观点，要明确标注
3. **考虑用户水平**：新手需要更多解释，老手需要更多数据
4. **永远提示风险**：每次分析结尾都要提醒投资风险
5. **不追热点**：不因为近期涨得好就推荐，要看长期逻辑
