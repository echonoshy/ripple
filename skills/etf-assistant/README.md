# ETF 投资助理 Skill

一个专业的 ETF 投资助手，支持行情查询、对比分析、定投计算等功能。

## 功能特性

### 核心功能
- **实时行情查询** - 查询 ETF 当前价格和涨跌幅
- **历史数据分析** - 查看近 1 周/1 月/3 月涨跌幅
- **批量查询** - 一次查询多个 ETF，快速对比
- **ETF 对比** - 对比两只 ETF 的关键指标
- **定投计算器** - 计算定投收益（假设年化 8%）
- **行业分类** - 按行业展示 ETF，提供投资建议
- **智能搜索** - 根据关键词搜索相关 ETF

### 技术特性
- ✅ 移除所有 ANSI 颜色代码和 emoji，解决乱码问题
- ✅ 自动依赖检查（curl, python3, bc）
- ✅ 请求超时和重试机制（10 秒超时，最多 2 次重试）
- ✅ 本地缓存（5 分钟 TTL），减少 API 请求
- ✅ 清晰的错误提示和诊断信息
- ✅ 修复 `get_etf_name()` bug
- ✅ 修复定投计算器公式（使用 Python 替代 bc）
- ✅ 移除 `head -100` 限制，避免 JSON 截断

## 使用方法

在 Ripple 中调用：
```bash
/etf-assistant <你的问题>
```

示例：
```bash
/etf-assistant 帮我查询沪深300的行情
/etf-assistant 对比一下科创50和创业板
/etf-assistant 如果每月定投1000元，10年后能有多少收益
/etf-assistant 有哪些科技类的ETF
```

## 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `list` | 显示常用ETF列表 | `list` |
| `price <代码>` | 查询实时行情 | `price 510300` |
| `history <代码>` | 查询历史涨跌幅 | `history 510300` |
| `batch <代码1> <代码2> ...` | 批量查询 | `batch 510300 159915 159919` |
| `hot` | 显示热门ETF | `hot` |
| `search <关键词>` | 搜索ETF | `search 芯片` |
| `compare <代码1> <代码2>` | 对比两只ETF | `compare 510300 159915` |
| `calc <代码> <金额> <年限>` | 定投计算器 | `calc 510300 1000 10` |
| `category` | 按行业分类展示 | `category` |
| `summary` | 投资摘要 | `summary` |
