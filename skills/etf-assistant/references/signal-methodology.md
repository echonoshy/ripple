# ETF 交易信号方法论

本文档记录信号系统的设计依据和评分标准。在解读信号结果或回答用户关于方法论的问题时参考。

## 一、五因子模型设计

信号系统采用五因子加权评分，总分 0-100。

### 因子权重

| 因子 | 权重 | 核心指标 | 设计依据 |
|------|------|----------|----------|
| 趋势 | 30% | MA5/MA20/MA60/MA120 排列、价格位置 | 趋势是最重要的方向性判断，决定大方向对错 |
| 动量 | 25% | RSI(14)、MACD(12,26,9) | 衡量价格变动速度和力度，捕捉拐点信号 |
| 波动 | 20% | 布林带(20,2)、ATR(14) | 衡量市场恐惧/贪婪程度，识别极端位置 |
| 量能 | 15% | 量比(20日)、量价配合 | 验证价格运动的有效性，无量则信号打折 |
| 形态 | 10% | KDJ(9,3,3)、连涨连跌 | 短期超买超卖辅助判断，权重低因易失效 |

### 权重设计原则
- 趋势最重要：跟随趋势的胜率远高于逆势操作
- 动量次之：RSI/MACD 能提供有效的入场时机
- 波动率第三：极端位置（布林带边界）有均值回归倾向
- 量能验证：成交量是"不会骗人"的指标，但单独使用意义有限
- 形态辅助：短期指标（KDJ）反应灵敏但噪音大，低权重

## 二、各因子评分细则

### 趋势因子 (0-100)

| 条件 | 分数 | 含义 |
|------|------|------|
| 价格 > MA5 > MA20 > MA60 | 85 | 完美多头排列 |
| 价格 > MA20 > MA60 | 72 | 标准多头排列 |
| 价格 > MA20 | 60 | 短期偏强 |
| 均线纠缠 | 50 | 方向不明 |
| 价格 < MA20 | 40 | 短期偏弱 |
| 价格 < MA20 < MA60 | 28 | 标准空头排列 |
| 价格 < MA5 < MA20 < MA60 | 15 | 完美空头排列 |

修正项：
- 价格位置 < 20%（接近低点）：+10
- 价格位置 > 80%（接近高点）：-10
- 跌破 MA120（半年线）：-5
- 站上 MA120：+5

### 动量因子 (0-100)

**RSI 部分**：
- RSI < 20：90分（极度超卖）
- RSI < 30：75分（超卖区）
- RSI 30-50：40分（偏弱）
- RSI 50-70：60分（偏强）
- RSI > 70：25分（超买区）
- RSI > 80：10分（极度超买）

**MACD 部分**：
- 金叉确认（柱由负转正）：85分
- 红柱放大：70分
- 红柱缩短：55分
- 绿柱缩短：45分
- 绿柱区域：30分
- 死叉确认（柱由正转负）：15分

两者取平均。

### 波动因子 (0-100)

**布林带位置**：
- 触及下轨 (< 10%)：80分（超卖反弹机会）
- 接近下轨 (10-25%)：65分
- 中间区域 (25-75%)：50分
- 接近上轨 (75-90%)：35分
- 触及上轨 (> 90%)：20分（超买回调风险）

修正项：
- 布林带宽度 < 3%：标注"变盘在即"
- ATR 突增 (> 1.5x 均值)：-10（波动加剧增加风险）

### 量能因子 (0-100)

| 场景 | 分数 |
|------|------|
| 放量上涨 (量比 > 2.0) | 80 |
| 温和放量上涨 (量比 > 1.5) | 70 |
| 缩量下跌 (量比 < 0.5) | 55 |
| 量能平稳 | 50 |
| 缩量上涨 (量比 < 0.5) | 40 |
| 放量下跌 (量比 > 1.5) | 30 |
| 恐慌放量下跌 (量比 > 2.0) | 20 |

### 形态因子 (0-100)

**KDJ**：
- J < 0（超卖）：80分
- K < D 低位：70分
- K > D：60分
- K < D：40分
- J > 80 高位：30分
- J > 100（超买）：20分

**连续涨跌**：
- 连涨 ≥ 4日：35分（警惕回调）
- 连跌 ≥ 4日：65分（可能反弹）

## 三、综合信号级别

| 总分区间 | 信号级别 | 建议仓位 | 含义 |
|---------|---------|---------|------|
| 80-100 | 强烈买入 | 80% | 多个因子共振看多 |
| 65-79 | 买入 | 60% | 大多数因子偏多 |
| 55-64 | 观望偏多 | 40% | 整体偏多但信号不强 |
| 45-54 | 观望 | 20% | 多空平衡，等待方向 |
| 25-44 | 卖出 | 10% | 大多数因子偏空 |
| 0-24 | 强烈卖出 | 0% | 多个因子共振看空 |

## 五、使用注意事项

1. **信号滞后性**：技术指标本质上是滞后指标，只能描述"已发生"，不能预测"将发生"
2. **单日信号不可靠**：应关注信号趋势（连续多日的变化），而非单日读数
3. **基本面缺失**：本系统纯技术面分析，不考虑估值、政策、宏观经济等基本面因素
4. **极端行情失效**：在政策驱动的急涨急跌中，技术指标可能全面失效
5. **组合使用**：建议将信号系统与 `risk.py` 风险预警、`portfolio.py` 组合诊断配合使用
